Xaheen Web Components Demo

Standards-compliant Web Components with Shadow DOM, CSS Custom Properties, and WCAG AAA accessibility.

Button Component

The <xaheen-button> component supports multiple variants, sizes, and states.

Variants

Primary Secondary Outline Ghost Destructive

Sizes

Extra Small Small Medium Large Extra Large

States

Normal Disabled Loading Full Width

Usage Example

<xaheen-button 
  variant="primary" 
  size="md"
  aria-label="Submit form"
>
  Submit
</xaheen-button>

Input Component

The <xaheen-input> component is form-associated and supports validation.

Basic Inputs

Input States

Form Integration

Submit Form Reset

Usage Example

<xaheen-input 
  label="Email Address"
  type="email"
  name="email"
  placeholder="your@email.com"
  required
  error="Please enter a valid email"
></xaheen-input>

Usage Example

<xaheen-card 
  variant="elevated" 
  clickable
  aria-label="View product details"
>
  <h4 slot="header">Product Name</h4>
  <p>Product description goes here</p>
  <div slot="footer">
    <xaheen-button size="sm">View Details</xaheen-button>
  </div>
</xaheen-card>
